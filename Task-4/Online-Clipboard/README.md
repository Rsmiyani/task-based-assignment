# Online Clipboard
 PHP, MySQL, HTML, CSS, and JavaScript app for sharing text with a random 4-digit code.

## Setup

1. Create the database by importing `database.sql` in MySQL.
2. Update `config.php` if your MySQL username, password, or database name is different.
3. Put the folder inside your XAMPP `htdocs` directory.
4. Open the app in your browser and save text to generate a 4-digit code.

## Features

- Save text and generate a random 4-digit code
- Fetch saved text using the code
- Copy fetched text with one click
- Minimal interface with Comic Sans style font
